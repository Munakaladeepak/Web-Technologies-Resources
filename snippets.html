The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, position and the actual content.
• Content - The content of the box, where text and images appear
• Padding - Clears an area around the content. The padding is transparent
• Border - A border that goes around the padding and content
• Margin - Clears an area outside the border. The margin is transparent
• Position: The position property specifies the type of positioning method used for an element (static, relative, fixed, absolute or sticky).

 In XML, namespaces are used to avoid naming conflicts when two or more different XML vocabularies (i.e., sets of element and attribute names) are used in the same document or set of documents.
 A namespace is identified by a unique URI (Uniform Resource Identifier), which is used to associate a set of element and attribute names with that namespace.
SAMPLE CODE:
<books xmlns:fiction="http://example.com/fiction"
xmlns:nonfiction="http://example.com/nonfiction">
<fiction:book id="001">
<fiction:author>John Doe</fiction:author>
<fiction:title>The Adventures of Tom Sawyer</fiction:title>
<fiction:genre>Children's fiction</fiction:genre>
<fiction:price>15.99</fiction:price>
</fiction:book>
<nonfiction:book id="002">
<nonfiction:author>Jane Smith</nonfiction:author>
<nonfiction:title>How to Build a House</nonfiction:title>

<nonfiction:price>29.99</nonfiction:price>
</nonfiction:book>
</books>

column1 datatype [optional_parameters],
column2 datatype [optional_parameters],
...
columnN datatype [optional_parameters],
[CONSTRAINT constraint_name] PRIMARY KEY (one_or_more_columns)
) ENGINE=InnoDB;
Sample code:
<?php
// Database configuration
$dbHost = "localhost";
$dbUsername = "username";
$dbPassword = "password";
$dbName = "database_name";
// Create database connection
$conn = mysqli_connect($dbHost, $dbUsername, $dbPassword, $dbName);
// Create table
$query = "CREATE TABLE users (
id INT(6) UNSIGNED AUTO_INCREMENT PRIMARY KEY,
firstname VARCHAR(30) NOT NULL,
lastname VARCHAR(30) NOT NULL,
email VARCHAR(50),
reg_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
)";
if (mysqli_query($conn, $query)) {
echo "Table users created successfully.";
} else {
echo "Error creating table: " . mysqli_error($conn);
}
// Close database connection
mysqli_close($conn);
?>
Output:
Table users created successfully.
ID firstname lastname email
INSERTING DATA INTO A TABLE:
• The INSERT INTO statement is used to add new records to a database table.
Syntax:
INSERT INTO table_name (column1, column2, ..., columnN)
VALUES (value1, value2, ..., valueN);
Sample code:
<?php
// Database configuration
$dbHost = "localhost";
$dbUsername = "username";
$dbPassword = "password";
$dbName = "database_name";
// Create database connection
$conn = mysqli_connect($dbHost, $dbUsername, $dbPassword, $dbName);
// Insert data
$firstname = "John";
$lastname = "Doe";
$email = "john.doe@example.com";
$query = "INSERT INTO users (firstname, lastname, email) VALUES ('$firstname', '$lastname', '$email')";
if (mysqli_query($conn, $query)) {
echo "New record created successfully.";
} else {
echo "Error: " . mysqli_error($conn);
}
// Close database connection
mysqli_close($conn);
?>
Output:
New record created successfully
(DESCRIPTION): In this code, the INSERT INTO statement is used to insert a new record into the "users" table. The VALUES keyword is used to specify the values for each column of the new record. The $firstname, $lastname, and $email variables are used to pass the values to the query. If the record is inserted successfully, the code will output "New record created successfully." Otherwise, it will output an error message. Note that this code is vulnerable to SQL injection attacks, so it's recommended to use prepared statements to prevent this.
CHANGING DATA OF A TABLE:
 The UPDATE statement is used to update existing records in a table.
Syntax:
UPDATE table_name
SET column1 = value1, column2 = value2, ..., columnN = valueN
WHERE condition;
Sample code:
<?php
// Database configuration
$dbHost = "localhost";
$dbUsername = "username";
$dbPassword = "password";
$dbName = "database_name";
// Create database connection
$conn = mysqli_connect($dbHost, $dbUsername, $dbPassword, $dbName);
// Update data
$id = 1;
$firstname = "John";
$lastname = "Doe";
$email = "john.doe@example.com";
$query = "UPDATE users SET firstname='$firstname', lastname='$lastname', email='$email' WHERE id=$id";
if (mysqli_query($conn, $query)) {
echo "Record updated successfully.";
} else {
echo "Error updating record: " . mysqli_error($conn);
}
// Close database connection
mysqli_close($conn);
?>
Output:
Record updated successfully
(DESCRIPTION):
In this code, the UPDATE statement is used to update the "firstname", "lastname", and "email" columns of a record in the "users" table. The SET keyword is used to specify the new values for each column. The WHERE clause is used to specify which record to update, in this case the record with an "id" of 1. If the record is updated successfully, the code will output "Record updated successfully." Otherwise, it will output an error message. Note that this code is vulnerable to SQL injection attacks, so it's recommended to use prepared statements to prevent this.

 The setcookie() function defines a cookie to be sent along with the rest of the HTTP headers.
 A cookie is often used to identify a user. A cookie is a small file that the server embeds on the user's computer. Each time the same computer requests a page with a browser, it will send the cookie too. With PHP, you can both create and retrieve cookie values.
Syntax:
• setcookie(name, value, expire, path, domain, secure, httponly);
HOW TO CREATE AND DELETE A COOLKE:
Creating cookie:
Syntax:
• setcookie(name, value, expiration);
Sample code:
<?php
$cookie_name = "user";
$cookie_value = "John Doe";
setcookie($cookie_name, $cookie_value, time() + (86400 * 30), "/"); // 86400 = 1 day
?>
<html>
<body>

if(!isset($_COOKIE[$cookie_name])) {
echo "Cookie named '" . $cookie_name . "' is not set!";

echo "Cookie '" . $cookie_name . "' isset!<br>";
echo "Value is: " . $_COOKIE[$cookie_name];
}
?>
</body>
</html>

HOW TO CREATE A SESSION:
 A session is a way to store information (in variables) to be used across multiple pages.
Syntax:
<?php
// Start the session
session_start();
?>
Sample code:
<!DOCTYPE html>
<html>
<body>
<?php
// Set session variables
$_SESSION["favcolor"] = "green";
$_SESSION["favanimal"] = "cat";
echo "Session variables are set.";
?>
</body>
</html>
Output:
Session variables are set.
HOW TO DESTROYE A SESSION:
 Destroying a PHP Session
 A PHP session can be destroyed by session_destroy() function. This function does not need any argument and a single call can destroy all the session variables. If you want to destroy a single session variable then you can use unset() function to unset a session variable.
Syntax:
<?php
session_start();
?>
Sample code:
<!DOCTYPE html>
<html>
<body>
<?php
// remove all session variables
session_unset();
// destroy the session
session_destroy();
echo "All session variables are now removed, and the session is destroyed."
?>
</body>
</html>
Output:
All session variables are now removed, and the session is destroyed.
